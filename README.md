SysPet - Sistema de Gestão para PetShop

Projeto desenvolvido na disciplina de Programação Estruturada em Python (Sistemas de Informação - FACISA).

Aluno: José Ivan Abrantes Virgínio
Instituição: FACISA - Sistemas de Informação

O projeto consiste em um sistema de gerenciamento de dados de um PetShop via linha de comando, estruturado em módulos para facilitar a organização das entidades.

Estrutura:
main.py: menu principal
clientes.py: módulo de tutores
pets.py: módulo de pets
veterinarios.py: módulo de veterinários
agendamentos.py: módulo de agenda
utils.py: funções utilitárias

Como rodar:
python main.py
